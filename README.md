Project Title: Vision-Based Control of Robotic Arm Mounted on Unmanned Surface Vehicle
📄 Project Description
This repository contains the design, simulation files, and source code supporting the research article: "Vision-Based Control of Robotic Arm Mounted on Unmanned Surface Vehicle for Rocket Recovery" published in IEEE Access.

This study presents the first fully autonomous system to combine a vision-based object detection algorithm, sliding mode trajectory tracking, and a robotic arm on a unmanned surface vehicle (USV) for direct rocket recovery in marine conditions.

Key Features:

The integration of YOLO-based object recognition, SMC-based trajectory tracking, and a 4-DOF SCARA robot arm enables precise localization, approach, and grasping of descending rocket stages under marine disturbances. The system offers a cost-effective and infrastructure-light approach and eliminates the need for human intervention, station-keeping thrusters, or large recovery platforms. Furthermore, this research uniquely combines AI-driven vision capabilities with advanced SMC techniques, paving the way for enhanced efficiency and reliability in rocket recovery operations in challenging oceanic environments.
